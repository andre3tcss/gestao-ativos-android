## 🎯 O Projeto
Aplicativo focado na usabilidade e eficiência para o cadastro, leitura e gerenciamento de hardwares, softwares e periféricos. O projeto adota princípios de Clean Architecture para separar as regras de persistência da interface de usuário.

## ⚙️ Arquitetura e Tecnologias
* **Linguagem:** Java nativo
* **Armazenamento:** SQLite (Offline-first) com padrão **DAO (Data Access Object)**
* **Interface:** Material Design 3 (ConstraintLayout, TextInputLayout)
* **Padrão de Projeto:** MVC / Clean Architecture adaptada (Camadas: `model`, `dao`, `database`, `view`)

## 🚀 Funcionalidades (MVP - Sprint 2)
